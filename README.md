This is a minimal Expo + React Native starter implementing **user authentication** using a modules-based folder structure and TypeScript.

## Quick start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the project:
   ```bash
   npm run start
   ```

3. Type check:
   ```bash
   npm run typecheck
   ```

## Notes
- Set `API_BASE_URL` in `src/core/api/client.ts` to `https://app.internsteps.com/api` or your API.
- Replace endpoints in `src/modules/auth/api/auth.ts` if your API paths differ.
